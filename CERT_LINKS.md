# Certificate Links (Vercel)

Once your site is deployed to Vercel, each certificate image is served from the project root. Replace `<YOUR_VERCEL_URL>` with your actual project URL (for example `https://protofilo.vercel.app`).

Example link format:

`https://<YOUR_VERCEL_URL>/<FILENAME>`

Replace `<YOUR_VERCEL_URL>` and use the filenames below.
