# Protofilo — Portfolio

Personal portfolio for Srinivas K — an aspiring backend developer (Java, Spring Boot, Cloud).

## Contents

- `index.html` — main portfolio page
- `styles.css` — styling
- `script.js` — site JS (mobile nav, smooth scroll, header effects)

## Quick start

Preview in a browser:

1. Open `index.html` directly.

Or run the local dev server (recommended):

```bash
npm install
npm run dev
# then open http://localhost:5173/
```

## Customize

- Edit contact email and social links in `index.html`.
- Replace content and assets in `public/` or `src/` as needed.
